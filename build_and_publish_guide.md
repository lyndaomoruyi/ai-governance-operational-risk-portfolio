# Build & Publish Guide

## Step 1: Create GitHub Repository
Create a public repository named `ai-governance-operational-risk-portfolio`.

## Step 2: Upload Files
Upload the full folder contents, including `index.html`, `styles.css`, `script.js`, `data/`, `docs/`, and `artifacts/`.

## Step 3: Enable GitHub Pages
Go to repository Settings > Pages. Choose the main branch and root folder. Save.

## Step 4: Add Link to Resume
Once published, add the GitHub Pages URL to the resume header and the Selected Projects section.

## Step 5: Interview Use
Use the website to walk employers through the business problem, AI workflow, risk controls, UAT testing, and resume relevance.
