# AI Operational Risk & Governance Command Center

A self-contained, black-and-white executive demonstration of AI-assisted CRM and third-party workflow automation.

## Files

- `index.html` — complete page structure and demo content
- `styles.css` — responsive black-and-white consulting-style layout
- `script.js` — all navigation, interactions, workflow simulation, governance dialogs, counters, and download behavior

## Run locally

1. Extract the ZIP.
2. Keep all three core files in the same folder.
3. Open `index.html` in Chrome, Edge, Firefox, or Safari.

No installation, build step, framework, server, or external library is required.

## Live demo flow

1. Open the **Live demo** section from the navigation.
2. Upload a file or select **Use sample operational record**.
3. Review the simulated AI-extracted fields.
4. Approve the fields.
5. Populate the CRM.
6. Synchronize the third-party platform.
7. Review and download the activity log.

The business impact figures are explicitly labeled as illustrative. Replace them with validated client baseline and pilot data before using them as factual claims.
