# AI Governance & Operational Risk Automation Portfolio

## Flagship Project
**Inventory Risk, Client Compliance & Replenishment Automation System**

This repository contains an employer-facing portfolio project demonstrating how generative AI can support IT-adjacent compliance, operational risk management, business analysis, and project delivery using synthetic operational data.

## What This Portfolio Demonstrates
- AI-assisted risk classification for inventory, client reporting, documentation, vendor follow-up, and escalation issues
- Compliance control mapping aligned to practical governance and audit-readiness concepts
- Business analysis artifacts including BRD, user stories, acceptance criteria, and UAT test cases
- Risk and action tracking workflows that reduce manual review and improve stakeholder visibility
- Human validation checkpoints for responsible AI use

## Core Artifacts
- `index.html` — portfolio website homepage
- `data/synthetic_inventory_risk_dataset.csv` — synthetic operational data
- `artifacts/ai_risk_automation_workbook.xlsx` — workbook with dashboard, risk register, control matrix, UAT, backlog, and sources
- `docs/` — project documentation, BRD, AI governance note, resume integration, and build guide

## Confidentiality Note
The portfolio uses synthetic data only. It is modeled on realistic commercial operations and compliance workflows, but it does not include confidential company, client, vendor, or account data.

## Research Basis
This portfolio structure follows common technical portfolio practices: a public project site, documented repository, work artifacts, and resume integration. It is informed by portfolio guidance from Harvard Career Services, MIT Career Advising & Professional Development, NIST AI RMF, and IIBA AI/business analysis guidance.
